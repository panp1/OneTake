# Workflow Artifacts

This folder contains reusable workflow artifacts for the current Centric Intake system.

Files:

- `end-to-end-workflow.mmd`
  Raw Mermaid source for the full intake-to-handoff process.
- `generation-pipeline.mmd`
  Raw Mermaid source for the worker generation pipeline and stage dependencies.
- `workflow-map.json`
  Structured machine-readable workflow map for LLMs, scripts, and future automation.

These artifacts are intended to be easy to ingest by:

- LLMs
- diagram renderers
- internal documentation tooling
- downstream automation or architecture indexing
