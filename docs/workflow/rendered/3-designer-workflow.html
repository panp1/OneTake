Amber Layer

Quick Edit — Flux 2

Photo-level retouching. Paint a mask over any area, describe the fix, and Flux 2 inpaints it. Great for fixing faces, removing artifacts, swapping backgrounds. Before/after comparison built in.

Green Layer

Graphic Editor — Overlay

Edit the text overlay live on the canvas. Change headlines, CTAs, font sizes, colors, positions. The daily driver for copy tweaks. Renders in real-time without regenerating the base image.

Regenerate

Seedream 4.5 — Full Regen

Start fresh with a new base image. Select what to change (scene, lighting, pose, expression, outfit) via checkboxes. Creates a compute job that runs Stage 2 or 4 again.

Figma Export

ZIP SVGs + Auto-Open

Downloads all creatives as SVGs in a ZIP file, organized by persona and version. Automatically opens the connected Figma file for drag-and-drop placement.

Figma Push

3-Level Push

Push creatives directly into Figma frames. Three granularity levels: entire campaign, single persona, or single version. Uses Nova frame naming convention.

Figma Sync

Bidirectional Status

Polling status bar detects when frames have been modified in Figma. Connect modal validates token and URL. Sync state persists across sessions.
