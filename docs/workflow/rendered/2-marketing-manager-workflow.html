Marketing Manager Workflow

The command center. Review AI-generated strategy, edit media budgets and targeting inline, manage landing page URLs, approve for designer, then give final sign-off to generate the agency package.

flowchart TD
    A["📥 Request lands on Dashboard\nstatus: review"]
    A --> B["Open Campaign Detail\n/intake/id"]

    B --> WS["Campaign Workspace — 3 tabs"]

    WS --> T1["📊 Campaign Brief"]
    T1 --> T1a["Messaging strategy\nvalue props, tone, hooks"]
    T1 --> T1b["Channel recommendations\nfrom cultural research"]
    T1 --> T1c["Persona summaries\ndemographics + psychology"]

    WS --> T2["📈 Media Strategy — EDITABLE"]
    T2 --> E1["✏️ Channel mix %\nclick → edit → auto-rebalance to 100%"]
    T2 --> E2["✏️ Ad set daily budgets\nclick $45 → type → blur saves"]
    T2 --> E3["✏️ Targeting interests\nadd/remove per tier (hyper/hot/broad)"]
    T2 --> E4["✏️ Split test variable\ndropdown: Creative / Audience / Placement"]
    T2 --> E5["✏️ Campaign objective\ndropdown: Lead Gen / Traffic / Conversions"]
    E1 & E2 & E3 & E4 & E5 --> SAVE["💾 Autosave\n800ms debounce → PATCH strategy_data JSONB"]

    WS --> T3["🌍 Regional Intelligence"]
    T3 --> T3a["Language, formality, dialect notes"]
    T3 --> T3b["Cultural guardrails"]

    B --> PC["Personas & Creatives Section"]
    PC --> PC1["Review composed creatives\nper persona, per channel"]
    PC --> PC2["✏️ Edit ad copy inline\nautosave per field"]
    PC --> PC3["HTML creative editor\nfull preview"]

    B --> LP["🟠 Landing Pages Card"]
    LP --> LP1["Enter Job Posting URL"]
    LP --> LP2["Enter Landing Page URL"]
    LP --> LP3["Enter AIDA Form URL"]
    LP --> LP4["View generated /lp/ URLs\nper persona"]

    B --> G1{"🚦 Marketing Approval Gate"}
    G1 -->|"Approve"| WAIT["Status → review\nawaiting designer"]
    G1 -->|"Request changes"| REGEN["Trigger regeneration\nfull pipeline or specific stage"]
    REGEN --> A

    WAIT --> DW["⏳ Designer works\nedits, retouches, Figma"]
    DW --> DS["Designer submits finals"]
    DS --> G2{"🚦 Designer Approval"}
    G2 -->|"Approved"| G3{"🚦 Final Approval Gate"}

    G3 -->|"Approve"| SENT["✅ Status → sent"]
    SENT --> MAGIC["🔗 Agency magic link generated\nauto-creates /agency/id URL"]
    MAGIC --> NOTIFY["📣 Teams + Outlook Notification\nsent to agency contact"]

Can Edit (Autosave)

  • Channel mix percentages — auto-rebalance
  • Ad set daily budgets — inline number input
  • Targeting interests — add/remove per tier
  • Split test variables — dropdown
  • Campaign objectives — dropdown
  • Ad copy — headline, body, CTA per creative
  • Landing page URLs — job posting, LP, AIDA

Controls (Actions)

  • Marketing approval — first gate
  • Final approval — after designer sign-off
  • Trigger pipeline regeneration (full or per-stage)
  • Agency magic link generation (auto on final approve)
  • View all pipeline stages and progress
  • Access admin panel (user management, schemas)
