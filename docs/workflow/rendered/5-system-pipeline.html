Full System Pipeline — All 6 Stages

Exact models, verification gates, and retry logic for each stage. Every model name matches the production config.

1
Strategic Intelligence
Kimi K2.5 — cultural research, web search
Qwen 3.5 397B — personas, brief, design direction (thinking mode)
Output: brief, personas, cultural research, campaign strategy, design direction
8-dimension eval gate (Qwen 3.5)
2
Character-Driven Images
Qwen 3.5 397B — actor descriptions
Seedream 4.5 — image generation (10 dimensions)
Flux 2 Pro — artifact cleanup (optional)
Qwen3-VL 8B — local VQA scoring
Output: actor profiles, base_image assets
VQA gate — face quality, hex artifacts, dirty rooms
3
Multilingual Ad Copy
Gemma 3 27B — ad copy (creative writing specialist)
Fallback: Kimi K2.5
Output: headlines, hooks, body, CTAs per persona × channel × language
Copy quality gate — brand voice, accuracy, tone
4
Layout Composition
Gemma 4 31B — graphic overlay copy
GLM-5 — HTML/CSS design generation
Playwright — headless render → PNG
Gemma 4 31B — creative VQA Phase 2
Flux 2 Pro — edit loop (up to 3 iterations)
Output: composed_creative, carousel_panel assets
VQA gate ≥ 0.85 — person visible, text legible, no artifacts
5
Video Generation
Gemma 4 31B — storyboard prompts
Seedream 4.5 — scene images
Kling V3 Omni — multi-shot video + lip sync
Alt: Sedeo 2.0 — UGC video
Output: 12-15s vertical UGC video
6
Landing Pages
Gemma 4 31B — LP copy (why, activities, FAQ)
GLM-5 — HTML page generation
Jinja2 — template rendering with hard facts
Output: per-persona HTML landing pages, 5 template variants
Drift validation — compensation, quals, URLs, work_mode

Verification Gates — Per Stage

Stage Gate Name Model What It Checks Pass Fail Action
1 Brief Evaluation qwen/qwen3.5-397b 8-dimension rubric: clarity, specificity, cultural fit, persona depth, messaging coherence, channel logic, budget rationale, brief completeness Score ≥ threshold Re-generate brief with feedback
2 Image VQA Qwen3-VL-8B (local) Face quality (no deformities), no hex color artifacts, no dirty rooms/pools, correct aspect ratio, person clearly visible All checks pass Retry with adjusted prompt (up to 3x), then Flux 2 cleanup
3 Copy Quality google/gemma-3-27b Brand voice compliance, factual accuracy (compensation, quals), tone match per persona, no AI slop Quality threshold met Retry with feedback, fallback to Kimi K2.5
4 Creative VQA Phase 2 google/gemma-4-31b Person visible in hero area, text legible (≤25% overlay), no rendering artifacts, brand consistency, CTR appeal, text zone safety Score ≥ 0.85 Flux 2 edit loop — inpaint problems (up to 3 iterations)
5 — — No automated gate (video quality reviewed by designer) — —
6 Drift Validation Deterministic (no LLM) Compensation matches intake, qualifications are verbatim subset, all CTA URLs match apply_url, work_mode sections correct, page lang attribute correct ALL facts match source Log error, mark evaluation_passed: false, skip upload
