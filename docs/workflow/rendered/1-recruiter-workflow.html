Recruiter Workflow

What the recruiter sees and does — from receiving a job description to sharing tracked links with candidates. The recruiter never touches strategy, raw assets, or pipeline stages. Their world is: submit → wait → share.

flowchart TD
    A["📋 Recruiter receives JD / RFP from hiring team"]
    A --> B{"🧙 5-Step Intake Wizard"}

    B -->|"Step 1"| B1["Upload RFP document\nor paste text"]
    B -->|"Skip"| B1a["Manual entry — no AI extraction"]
    B1 --> B2["Gemma 4 extracts fields\ntask type, regions, quals, budget"]
    B1a --> B3
    B2 --> B3["Step 2: Select Task Type\n+ Work Mode (onsite/remote)"]
    B3 --> B4["Step 3: Project Details\ntitle, volume, regions, languages, budget"]
    B4 --> B5["Step 4: Requirements\nqualifications, engagement, AIDA form URL"]
    B5 --> B6["Step 5: Review & Submit\nall fields visible, edit buttons per section"]
    B6 --> C["🚀 Request Created\nstatus: draft → generating"]

    C --> D["⚡ Pipeline auto-triggers\n6-stage generation begins"]
    D --> E["⏳ Dashboard shows progress\nstage indicators update in real-time"]
    E --> F["📣 Teams + Outlook Notification\n'Creatives ready for review'"]

    F --> G["📂 Recruiter opens Creative Library\n/intake/id or /r/campaign-slug"]

    G --> H["🖼️ Composed Creatives\nfiltered by channel tabs\nMeta · LinkedIn · TikTok · Google"]
    G --> I["🔗 Sticky Link Builder\nalways visible on right side"]
    I --> I1["Select base URL\nlanding page, job posting, or AIDA form"]
    I1 --> I2["Auto-generates UTM short link\ncampaign + source + medium + content"]
    I2 --> I3["📋 Copy link\nshare with candidates via email/social"]

    G --> LP["🟠 Landing Pages\nper-persona /lp/ URLs visible"]
    LP --> LP1["Copy landing page URL\nuse as ad destination"]

    G --> J["⬇️ Download creatives\nindividual or batch"]

Has Access To

  • 5-step intake wizard with AI pre-fill
  • Dashboard with request status tracking
  • Creative Library — composed creatives only
  • UTM Link Builder — tracked short links
  • Generated landing page URLs
  • Creative downloads (individual + batch)
  • Teams + Outlook email notifications

No Access To

  • Media strategy editing (budgets, interests)
  • Raw/base actor images
  • Pipeline stages or regeneration
  • Designer tools (edit, Figma)
  • Agency portal or magic links
  • Admin panel or user management
  • Approval controls
